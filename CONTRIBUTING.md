# Contributing to Auto Job Apply

We welcome contributions from students, developers, and everyone interested in making job hunting easier!

## Code of Conduct

Please read and follow our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) to ensure our community remains welcoming.

## How to Contribute

### Reporting Bugs
1. Check if the bug already exists in [Issues](../../issues)
2. Create a new issue with:
   - Clear title describing the bug
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots if applicable

### Suggesting Features
1. Check if the feature is already requested
2. Open an issue with label `enhancement`
3. Describe the feature and why it would be helpful

### Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/amazing-feature`
3. Make your changes
4. Test your changes: `npm run test`
5. Commit with clear messages: `git commit -m 'Add amazing feature'`
6. Push: `git push origin feature/amazing-feature`
7. Open a Pull Request with description of changes

## Development Setup

```bash
git clone https://github.com/yourusername/auto-job-apply.git
cd auto-job-apply
npm install
npm run dev
```
